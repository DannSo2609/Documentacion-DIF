🔄 ¿Qué pasa después?
  • Los documentos quedan asociados al embarque y visibles para:
  • Tráfico
  • Aduanas
  • Finanzas
  • Coordinación logística
  • El sistema impide liberar la carga si faltan documentos obligatorios.
🟡 Notas importantes
  • Sin BL, no se puede liberar el embarque. Asegúrate de cargar el archivo tan pronto esté disponible.
  • El sistema puede mostrar advertencias si intentas aprobar embarques incompletos.
  • Algunos documentos como el certificado de origen solo aplican si lo solicita aduanas.
✅ Buenas prácticas
  • Renombra los archivos antes de cargarlos:
    Ejemplo: Factura Cliente 20Jun2025.pdf
  • Si un documento no está disponible aún, deja una nota interna para el equipo de tráfico.
  • No cargues documentos personales ni archivos innecesarios.
